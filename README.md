# Портфолио ГИС-специалиста

Веб-сайт для презентации картографических проектов и макетов, выполненных в QGIS.

### Ключевые навыки и примененные технологии:
* **Геоинформационный анализ:** работа с растровыми и векторными слоями в среде QGIS, пространственная привязка данных по опорным точкам.
* **Векторная картография:** ручная оцифровка административных границ по геодезическим ориентирам, управление базами данных и таблицами атрибутов.
* **Картографический дизайн:** качественная классификация объектов, устранение визуального шума базовых подложек (OpenStreetMap/Google) с помощью монохромной генерализации, настройка динамической кластеризации точек с кастомной SVG и шрифтовой символикой.
* **Веб-разработка:** адаптивная верстка интерфейса (HTML5, CSS3, Flexbox Grid) для корректного отображения портфолио на любых устройствах.
